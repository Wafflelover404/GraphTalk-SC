üìã Integration Guide

1 Add Configuration Script

Add this before the widget script to configure the chatbot:

<script> window.GraphTalkChatbot = { apiKey: 'wk_N7WsBd1_cGsbgLI7DFihlK5KfLO80pxI5JZ9uqH1mwE', catalogId: 'demo-catalog', apiEndpoint: 'http://localhost:9001', position: 'bottom-right' // or 'bottom-left' }; </script>

2 Load Widget Script

Add the widget script just before closing </body> tag:

<script src="opencart_chatbot_widget.js"></script>

3 Configure Your Store

Update the configuration parameters:

  • apiKey: Your GraphTalk API key
  • catalogId: Your OpenCart catalog ID
  • apiEndpoint: GraphTalk API endpoint URL
  • deepseekApiKey: DeepSeek API key for LLM (optional)
  • position: 'bottom-right' or 'bottom-left'

4 Test the Widget

Open your OpenCart store and click the chatbot bubble to test.

You should see the AI assistant ready to help with product searches.

üì° API Endpoint Used

Endpoint: POST /api/v1/catalog/{catalog_id}/ai-overview

The widget sends user queries to this endpoint which returns AI-generated overviews of the top 5 most relevant products.

Request Body:

{ "query": "user search query", "limit": 20, "min_relevance": 0.2 }

Response:

{ "overview": "AI-generated product overview text..." }
